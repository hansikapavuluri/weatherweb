A modern, fully-responsive weather website built with Django that displays real-time weather information with beautiful animations and a sleek glassmorphism interface.

## ✨ Features

- **Real-time Weather Data** - Fetches current weather from OpenWeatherMap API
- **Dark/Light Theme Toggle** - Seamless theme switching with CSS variables and localStorage
- **Glassmorphism UI** - Modern frosted glass design with smooth transitions
- **Fully Responsive** - Perfect experience on mobile, tablet, and desktop
- **Django Best Practices** - Proper template structure, static files, and CSRF protection

## 🛠️ Technologies Used

- **Backend**: Django 4.x
- **Frontend**: HTML5, CSS3 (Flexbox & Grid), Vanilla JavaScript
- **API**: OpenWeatherMap API
- **Animations**: Pure CSS keyframes and transitions

## 📱 Key Components

### Weather Display
- City name with location icon
- Large temperature display (°C)
- Weather description
- Dynamic animated icons (rain, sun, clouds)

### Weather Details Grid
- 💧 Humidity
- 🌬️ Wind Speed
- ⚖️ Pressure
- 🌡️ Feels Like
